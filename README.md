30 DAYS RECIPES
“30 DAYS RECIPES” merupakan aplikasi Android sederhana yang berisi tips resep masakan untuk 30 hari ke depan. Aplikasi ini menampilkan daftar resep harian yang dapat dilihat pengguna dengan tampilan yang sederhana dan mudah dipahami. Proyek ini saya buat sebagai media latihan dalam memahami dasar rancangan antarmuka menggunakan Jetpack Compose.
